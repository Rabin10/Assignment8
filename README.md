# Rock–Paper–Scissors (React + Vite)

This Assignment is a Rock–Paper–Scissors game built using React, Vite, components, props, state, and event handling.  
The user selects rock, paper, or scissors, and the computer animates its selection before revealing the result.


# Setup Instructions

# 1. Clone the repository
open the terminal
git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>

# 2. Install dependencies
npm install

# 3. Run the development server
npm run dev

The app will be available at the URL shown in your terminal (usually it is http://localhost:5173).
